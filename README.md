# FileSizeFinder
Scans the file from external storage and displays the list of 10 biggest files, average file size and 5 most frequent extensions

* [ ] Displays list of name and size of ten biggest files.
* [ ] Dispalys average size the files and five most frequent extentions with their frquencies.
* [ ] Notifies the user about scanning and its progress.
